# CyberMind AI - Data Directory

This directory contains all datasets, reference files, blacklist information, whitelist information, testing samples, and static resources required by CyberMind AI.

## Folder Structure

### blacklist/
Contains malicious URLs, domains, IPs, hashes and phishing keywords.

### whitelist/
Contains trusted domains, URLs, IPs and email providers.

### phishing/
Contains Machine Learning datasets for phishing detection.

### email/
Contains common email providers, free email providers, corporate domains and role-based email names.

### disposable_email/
Contains temporary and disposable email providers.

### domain/
Contains trusted TLDs, suspicious TLDs, registrar information and country TLD references.

### ip/
Contains reserved IP ranges, private IP ranges, suspicious IP samples and country mappings.

### qr/
Contains safe and phishing QR code samples for testing.

### files/
Contains MIME types, dangerous extensions, safe extensions and malware hash samples.

### security/
Contains security headers, SSL cipher information and common network ports.

### samples/
Contains sample URLs, emails, domains, IP addresses, QR URLs and filenames for testing.

### cache/
Temporary cache files generated by the application.

---

All datasets in this folder are either:

- Publicly available
- Open-source
- Self-created
- Used only for educational and research purposes
